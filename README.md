### FRONTEND
Ten en cuenta estas versiones
*  Angular CLI: 20.3.3
*  Node: 24.9.0
*  Package Manager: npm 11.6.0
En angular hay ciertas estructuras, considera lo siguiente
- Components -> Son elementos pequeños reutilizables (botones, tarjetas, navbar, etc).
- Pages -> Pantallas principales (login, resgistro, perfil, etc).
- Services -> Son llamadas HTTP al backend, por ejemplo, usuario.service.ts para crear un usuario.
- models -> Las clases o interfaces que representen a las entidades, por ejemplo, usuario.model.ts
